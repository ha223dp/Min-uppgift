CSS

Vad är en regel i CSS och vad består den av? Css-regler finns i två delar. De delarna anvisas på vilka element som används, och vilka egenskaper som det man vill helst ha förändrat. Selektorn väljer ut vilka specifika element som bestäms, som därmed vilka element som ska påverkas av CSS-regeln.

CSS-specifikationen berättar hur ett prioriterat system kan avgöra vilken stilregel som ska gälla och hur regeln matchar mot ett visst element. I det här sammanhanget så kallas det en kaskad, där prioriteringar ligger efter beräkningar till hur de regler som tilldelats för ett förutsägbart resultat.
